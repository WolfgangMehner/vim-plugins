Installation

Install the plug-in using a plug-in manager or by unpacking the archive inside the folder $HOME/.vim (Linux) or $HOME/vimfiles (Windows). Then make the plug-in help available by rebuild the helptags. The argument is the folder doc inside the directory where the plug-in is installed, e.g.: 

:helptags $HOME/.vim/doc/

Make sure there is a file $HOME/.vimrc (Linux) or $HOME/_vimrc (Windows) with the following line, so that the plug-ins are loaded: 

filetype plugin on

The configuration variables listed below also go into the vimrc file, e.g.: 

dodobob
let g:Awk_InsertFileHeader = 'no'
let g:BASH_InsertFileHeader = 'no'
let g:C_InsertFileHeader = 'no'
let g:Latex_InsertFileProlog = 'no'
let g:Lua_InsertFileHeader = 'no'
let g:Perl_InsertFileHeader = 'no'
let g:Vim_MapLeader = '#'
dodobob

Documentation

The documentation of the plug-in is available through the built-in help system: 

dodobob
:help awk-support
:help bash-support
:help c-support
:help latex-support
:help lua-support
:help perl-support
:help vim-support

Help is also avaiable for all configuration options and ex-commands: 

dodobob
:help
:help g:BASH_Executable
:help :BashOutputMethod
:help
:help g:Latex_Typesetter
:help :LatexProcessing
:help g:Lua_Executable
:help :LuaOutputMethod
:help
:help g:Vim_CreateMapsForHelp

Configuration Options

Set the mapleader used by the plug-in: 

dodobob
let g:Awk_MapLeader = '#'
let g:BASH_MapLeader = '#'
let g:C_MapLeader = '#'
let g:Latex_MapLeader = '#'
let g:Lua_MapLeader = '#'
let g:Perl_MapLeader = '#'
let g:Vim_MapLeader = '#'
dodobob

Now all maps start with the character #, e.g. to comment out some lines of code use #cc.

External Programs

not applicable

Set the script interpreter: 

dodobob
let g:Awk_Executable = '/usr/local/bin/awk'
let g:BASH_Executable = '/usr/local/bin/bash'
let g:Perl_Executable = '/usr/local/bin/perl'

Configure the path where to find all the executables: 

let g:Lua_BinPath = $HOME.'/lua52/bin'

Or set the script interpreter and compiler individually: 

let g:Lua_Executable   = '/usr/local/bin/lua'
let g:Lua_CompilerExec = '/usr/local/bin/luac'

Because of the lack of a standard, the extension for compiled files can be changed as well: 

let g:Lua_CompiledExtension = 'lc'

Choose the typesetter used on start-up: 

let g:Latex_Typesetter = 'luatex'
